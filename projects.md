# Proyectos de análisis de datos

Cada proyecto está orientado a **resultados de negocio**: ahorro de costes, eficiencia operativa o mejor toma de decisiones.

---

## Proyecto 1 – Optimización de costes de inventario

> **Objetivo:** reducir costes asociados a stock inmovilizado y sobreinventario.

**Descripción ejecutiva**  
Análisis de varios años de datos de inventario por producto, categoría y almacén para detectar stock sobrante, rotación lenta y oportunidades de consolidación.

**Técnicas y herramientas analíticas**

- Python (Pandas) para limpieza y análisis exploratorio de datos
- Cálculo de indicadores: rotación de inventario, días de stock, cobertura
- Segmentación por categoría, almacén y proveedor
- Visualización de resultados en Power BI / Tableau

**Beneficio conseguido / esperado**

- Identificación de productos con sobrestock valorados en **X€**
- Propuesta de reducción de stock que supondría un ahorro potencial del **X%** en costes de inventario
- Base de análisis recurrente para decisiones de compra y logística

## Proyecto 2 – Dashboard de KPIs financieros

> **Objetivo:** dar visibilidad a dirección sobre márgenes, costes y rentabilidad por línea de negocio.

**Descripción ejecutiva**  
Diseño y construcción de un dashboard financiero interactivo que unifica datos de diferentes fuentes (ventas, costes, gastos) en una única vista.

**Técnicas y herramientas analíticas**

- SQL para extracción y unión de datos de distintas tablas
- Modelado de datos en Power BI / Tableau
- Creación de medidas (márgenes, variaciones, % sobre ingresos)
- Segmentación por producto, región y canal

**Beneficio conseguido / esperado**

- Reducción del tiempo de creación de i
